How to make BOM images

1: Use the print feature on Onshape to render the image. Then select "Print to PDF".
2: Take the rendered PDF and open it in GIMP2.
3: Change the resolution from 100.00 to 1000.00 when prompted on open.
4: Apply the silk screen to the Fusion Controller.
5: Autocrop the image to eliminate the extra white space.
5: Export the new high resolution pdf file (Overwrite the exsisting).
6: Then export the same image to a .png file.
